================================================================================
  PROJECT THỰC TẾ 2: E-COMMERCE DASHBOARD
================================================================================

THỜI LƯỢNG ƯỚC TÍNH:
  - 6-10 giờ (stretch project)

================================================================================
  CHEATSHEET NHANH
================================================================================

  - Dashboard này gom đủ auth + guard + interceptor + CRUD products
  - Product management = reactive form + table/search/filter
  - Xem đây là stretch project sau khi bạn đã xong core 14 ngày


📌 MỤC TIÊU: Xây dựng dashboard quản lý sản phẩm
   Project nâng cao, áp dụng toàn bộ kiến thức!

================================================================================
  YÊU CẦU CHỨC NĂNG:
================================================================================

  1. AUTHENTICATION
     - Login page
     - Auth guard bảo vệ routes
     - JWT token handling (interceptor)

  2. DASHBOARD
     - Tổng quan: cards thống kê (total products, orders, revenue)
     - Charts (dùng ng2-charts hoặc chart.js)

  3. PRODUCT MANAGEMENT (CRUD)
     - Danh sách sản phẩm (table + pagination + sort)
     - Thêm/Sửa sản phẩm (reactive form)
     - Xóa sản phẩm (confirm dialog)
     - Search + Filter (RxJS debounce)

  4. ROUTING
     - /login
     - /dashboard (guard protected)
     - /dashboard/products
     - /dashboard/products/:id
     - /dashboard/orders
     - 404 page

  5. RESPONSIVE
     - Sidebar navigation (collapse trên mobile)
     - Responsive table/grid

================================================================================
  KIẾN THỨC ÁP DỤNG:
================================================================================

  ✅ Components: 10+ components
  ✅ Services: AuthService, ProductService, OrderService
  ✅ Routing: nested routes, lazy loading, guards
  ✅ Forms: Reactive forms với validation
  ✅ HTTP: CRUD operations, interceptors
  ✅ RxJS: search, combine filters, error handling
  ✅ State: Signals hoặc service-based state
  ✅ Material: dialog, table, snackbar, sidenav, toolbar
  ✅ Guards: auth guard, admin guard
