📌 BÀI 2: NULL SAFETY
================================================================================

  Tạo file bai2.ts:

  1. Khai báo:
     let userName: string | null = null;
     let userAge: number | undefined = undefined;

  2. Viết function greet nhận tham số nullable:
     function greet(name: string | null): string {
       // Dùng ?? và ?. để xử lý null
       // So sánh với cách viết trong Dart
     }

  3. Thử dùng non-null assertion (!):
     let element = document.getElementById('app')!;
     // Khi nào nên dùng? Khi nào KHÔNG nên dùng?

📌 BÀI 3: UNION TYPES VÀ LITERAL TYPES
================================================================================

  Tạo file bai3.ts:

  1. Viết function formatId nhận id là string HOẶC number:
     function formatId(id: string | number): string {
       // Dùng typeof để kiểm tra kiểu (type narrowing)
       // typeof id === 'string' → ...
       // typeof id === 'number' → ...
     }

  2. Tạo type cho trạng thái đơn hàng (dùng literal types):
     type OrderStatus = 'pending' | 'processing' | 'shipped' | 'delivered';

     // Viết function getStatusMessage(status: OrderStatus): string
     // Dùng switch-case cho từng trạng thái

  3. So sánh: Trong Dart bạn viết bài này thế nào? (dùng enum?)
