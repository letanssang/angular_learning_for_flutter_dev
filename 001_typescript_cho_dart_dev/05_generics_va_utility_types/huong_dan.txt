================================================================================
  1. NHUNG THU GIAI QUYET DUOC VIEC NGAY
================================================================================

  interface User {
    id: number;
    name: string;
    email: string;
    avatar?: string;
  }

  type UpdateUser = Partial<User>;
  type UserPreview = Pick<User, 'id' | 'name'>;
  type CreateUser = Omit<User, 'id'>;
  type UserMap = Record<string, User>;
  type FrozenUser = Readonly<User>;

  💡 Neu ban thay nhung type nay trong codebase, chi can doc nhu sau:
  - Partial<T> = tat ca field thanh optional
  - Pick<T, K> = lay mot vai field
  - Omit<T, K> = bo mot vai field
  - Record<K, V> = object map
  - Readonly<T> = khong cho sua

================================================================================
  2. CONDITIONAL TYPES - DOC DUOC LA DU
================================================================================

  type IsString<T> = T extends string ? 'yes' : 'no';

  type A = IsString<string>;  // 'yes'
  type B = IsString<number>;  // 'no'

  Khi gap loai nay trong du an:
  - dung hoang
  - doc tu trong ra ngoai
  - thu thay T bang type cu the de hieu ket qua

================================================================================
  3. MAPPED TYPES - BIEN DOI HANG LOAT
================================================================================

  type Optional<T> = {
    [K in keyof T]?: T[K];
  };

  type FormState<T> = {
    [K in keyof T]: {
      value: T[K];
      error: string | null;
      touched: boolean;
    };
  };

  Ung dung thuc te:
  - tao wrapper state cho model
  - tao helper types cho form va cache layer
