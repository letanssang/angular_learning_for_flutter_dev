  APPENDIX - GENERICS VÀ UTILITY TYPES NÂNG CAO
================================================================================

THỜI LƯỢNG ƯỚC TÍNH:
  - 20-30 phút

📌 MỤC TIÊU:
  - Mở rộng sau khi bạn đã xong core 2 tuần
  - Hiểu thêm về utility types nâng cao trong codebase lớn
  - Biết đọc mapped types và conditional types mà không cần thuộc lòng

  ⚠️ KHÔNG BẮT BUỘC để:
  - tạo feature CRUD cơ bản
  - dùng router, forms, HTTP, signals
  - hoàn thành Todo app trong lộ trình cấp tốc

================================================================================
  1. NHỮNG THỨ GIẢI QUYẾT ĐƯỢC VIỆC NGAY
================================================================================

  interface User {
    id: number;
    name: string;
    email: string;
    avatar?: string;
  }

  type UpdateUser = Partial<User>;
  type UserPreview = Pick<User, 'id' | 'name'>;
  type CreateUser = Omit<User, 'id'>;
  type UserMap = Record<string, User>;
  type FrozenUser = Readonly<User>;

  💡 Nếu bạn thấy những type này trong codebase, chỉ cần đọc như sau:
  - Partial<T> = tất cả field thành optional
  - Pick<T, K> = lấy một vài field
  - Omit<T, K> = bỏ một vài field
  - Record<K, V> = object map
  - Readonly<T> = không cho sửa

================================================================================
  2. CONDITIONAL TYPES - ĐỌC ĐƯỢC LÀ ĐỦ
================================================================================

  type IsString<T> = T extends string ? 'yes' : 'no';

  type A = IsString<string>;  // 'yes'
  type B = IsString<number>;  // 'no'

  Khi gặp loại này trong dự án:
  - đừng hoảng
  - đọc từ trong ra ngoài
  - thử thay T bằng type cụ thể để hiểu kết quả

================================================================================
  3. MAPPED TYPES - BIẾN ĐỔI HÀNG LOẠT
================================================================================

  type Optional<T> = {
    [K in keyof T]?: T[K];
  };

  type FormState<T> = {
    [K in keyof T]: {
      value: T[K];
      error: string | null;
      touched: boolean;
    };
  };

  Ứng dụng thực tế:
  - tạo wrapper state cho model
  - tạo helper types cho form và cache layer
