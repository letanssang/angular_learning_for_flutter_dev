  TYPESCRIPT CHO DART DEV - INTERFACE, TYPE VA GENERICS CO BAN
================================================================================

THOI LUONG UOC TINH:
  - 30-40 phut

📌 MUC TIEU:
  - Hieu interface va type de doc model/service/component Angular
  - Biet 3 utility thuc dung nhat: Partial, Pick, Omit
  - Nam generics co ban de doc HttpClient va reusable helpers

  ⚠️ Trong lo trinh 2 tuan:
  - Bai nay la CORE
  - Chi can du de doc va sua code Angular thuc
  - Utility types nang cao, conditional types, mapped types de sau

================================================================================
  1. INTERFACE TRONG TYPESCRIPT
================================================================================

  // ──── DART ────
  abstract class Animal {
    String name;
    void speak();
  }

  // ──── TYPESCRIPT ────
  interface Animal {
    name: string;
    speak(): void;
  }

  ⚠️ KHAC BIET QUAN TRONG:
  - TS interface chi ton tai o compile-time, runtime se bi xoa
  - Interface dung de mo ta "shape" cua object
  - Angular dung no rat nhieu cho models, API contracts, DTOs

  class Dog implements Animal {
    name: string;

    constructor(name: string) {
      this.name = name;
    }

    speak(): void {
      console.log(`${this.name} says Woof!`);
    }
  }

================================================================================
  2. TYPE ALIAS
================================================================================

  type Point = {
    x: number;
    y: number;
  };

  type ID = string | number;
  type Callback = (data: string) => void;

  Khi nao dung gi?
  - interface: model object, API response, public contract
  - type: union, function type, alias nhanh, utility type

================================================================================
  3. UNION, INTERSECTION, OPTIONAL, READONLY
================================================================================

  type Status = 'idle' | 'loading' | 'error';

  type HasName = { name: string };
  type HasAge = { age: number };
  type Person = HasName & HasAge;

  interface User {
    readonly id: number;
    name: string;
    email?: string;
  }

  const user: User = { id: 1, name: 'An' };
  user.name = 'Binh';   // OK
  user.id = 2;          // ERROR

  So sanh Flutter:
  - union type = cai Dart khong co san
  - readonly ~ final
  - optional property ~ String? email

================================================================================
  4. GENERICS CO BAN
================================================================================

  interface ApiResponse<T> {
    data: T;
    message: string;
  }

  interface Paginated<T> {
    items: T[];
    total: number;
  }

  const productResponse: ApiResponse<Paginated<Product>> = {
    data: {
      items: [],
      total: 0,
    },
    message: 'OK',
  };

  // Angular se gap rat nhieu:
  // http.get<Product[]>('/api/products')
  // signal<Todo[]>([])
  // input<User>()

  💡 Cach doc:
  - <T> = cho type vao
  - ApiResponse<Product[]> = response co data la danh sach Product

================================================================================
  5. 3 UTILITY TYPES DU DUNG NHAT
================================================================================

  interface Product {
    id: number;
    name: string;
    price: number;
    description: string;
  }

  type CreateProduct = Omit<Product, 'id'>;
  type UpdateProduct = Partial<Product>;
  type ProductPreview = Pick<Product, 'id' | 'name' | 'price'>;

  Thuc te trong app:
  - Omit: payload tao moi
  - Partial: payload cap nhat
  - Pick: data nhe cho list/card

================================================================================
  6. DOC TYPE ERROR THUONG GAP
================================================================================

  Error 1:
  Type 'string | undefined' is not assignable to type 'string'

  Nghia la:
  - gia tri co the undefined
  - nhung noi dang dung chi chap nhan string

  Cach xu ly:
  - check if truoc
  - dat default value
  - doi type neu domain thuc su cho phep undefined

  Error 2:
  Property 'x' does not exist on type 'Y'

  Nghia la:
  - ban dang doc sai shape cua object
  - can kiem tra interface/type cua data

  Error 3:
  Type 'A' is not assignable to type 'B'

  Nghia la:
  - payload, model hoac response dang khong dung contract
